# Glosario Legal Normalizado — LIA

> **Uso**: Inyectado al Auditor y Financial.
> **Propósito**: Terminología consistente y niveles de severidad estandarizados.

---

## 1. Términos Contractuales Fundamentales

| Término | Definición | Contexto de Uso |
| :--- | :--- | :--- |
| **Objeto** | La prestación o conjunto de prestaciones que las partes se obligan a cumplir. | El Intake verifica que el objeto esté claramente definido. |
| **Causa** | El motivo o razón jurídica que justifica la obligación. Debe ser lícita. | El Auditor verifica que la causa no sea ilícita (Art. 1524 C.C.). |
| **Consentimiento** | Manifestación de voluntad libre, consciente e informada de las partes. | El Auditor detecta vicios del consentimiento. |
| **Capacidad** | Aptitud legal de las partes para obligarse. Personas jurídicas: representante legal con facultades. | El Intake verifica que firmantes tengan capacidad. |
| **Contraprestación** | Lo que cada parte recibe a cambio de su obligación. | Elemento clave en contratos bilaterales. |
| **Plazo** | Período de tiempo para el cumplimiento de las obligaciones. | El Intake verifica que exista y sea razonable. |

## 2. Tipos de Cláusulas

| Tipo | Definición | Implicación |
| :--- | :--- | :--- |
| **Esenciales** | Sin ellas, el contrato no existe o degenera en otro tipo. | **CRITICAL** si faltan. |
| **Naturales** | Se incorporan por ley aunque no se pacten expresamente. | Se aplican por defecto. |
| **Accidentales** | Agregadas por acuerdo de las partes. No se presumen. | Requieren pacto expreso. |
| **De adhesión** | Redactadas unilateralmente; la otra parte solo acepta o rechaza. | Aplica *contra proferentem*. |
| **Abusivas** | Generan desequilibrio significativo e injustificado. | **CRITICAL**. Citar Ley 1480 si aplica. |

## 3. Niveles de Severidad (FUENTE CANÓNICA)

Todos los agentes DEBEN usar exclusivamente estos niveles:

| Nivel | Significado | Acción Requerida |
| :--- | :--- | :--- |
| **CRITICAL** | Riesgo de nulidad, ilegalidad o pérdida patrimonial grave. | **No firmar** sin resolver. |
| **HIGH** | Riesgo legal significativo o exposición financiera importante. | **Negociar** antes de firmar. |
| **MEDIUM** | Omisión relevante o práctica no recomendada, pero no ilegal. | **Recomendar** mejora. |
| **LOW** | Observación menor. Mejora de forma o claridad. | **Sugerir** mejora. |
| **INFO** | Información contextual. No es un riesgo. | **Documentar** para referencia. |

### Criterios de Clasificación

- **CRITICAL**: Viola norma imperativa, riesgo patrimonial > 30% del valor, ilicitud en objeto/causa, faltan elementos esenciales.
- **HIGH**: Cláusula desproporcionada pero no ilegal, falta protección legal importante, exposición a sanciones regulatorias.
- **MEDIUM**: Falta cláusula recomendada pero no obligatoria, redacción ambigua.
- **LOW**: Errores de forma, inconsistencias menores.

## 4. Glosario de Términos Frecuentes

| Término | Definición |
| :--- | :--- |
| **Cláusula penal** | Penalidad pactada para caso de incumplimiento. Sujeta a límites legales. |
| **Póliza de cumplimiento** | Seguro que garantiza el cumplimiento de obligaciones contractuales. |
| **Otro sí** | Documento modificatorio. Prevalece sobre el texto inicial en lo que modifique. |
| **Cláusula compromisoria** | Pacto que somete las disputas a arbitraje. |
| **Fuerza mayor** | Evento imprevisible e irresistible que impide cumplimiento (Art. 64 C.C.). |
| **Indemnización** | Compensación económica por daños de incumplimiento. |
| **Lucro cesante** | Ganancia que se dejó de percibir por incumplimiento. |
| **Daño emergente** | Pérdida patrimonial directa por incumplimiento. |
| **Cesión** | Transferencia de derechos/obligaciones a un tercero. |
| **Confidencialidad (NDA)** | Obligación de no divulgar información reservada. |
